# SYNAPSE: Local AI Workspace Assistant

SYNAPSE is a local-first, privacy-respecting desktop application that acts as a private knowledge operating system for your computer. It allows you to search semantically and chat with your local documents entirely offline.

## 🚀 Target Features
*   Local-first document parsing (PDF, Word, Excel, Text)
*   Semantic search using local vector embeddings
*   Offline LLM integration via Ollama
*   Native PySide6 Desktop GUI
